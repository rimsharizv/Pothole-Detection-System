# Pothole Detection System
Authors: Rimsha Rizvi & Shehriar Burney

## Description
The Pothole Detection System (PDS) is an Android mobile application designed to enhance the travel experience of users by integrating Google Maps API with a backend database to deliver up-to-date information on the locations of potholes and other potential hazards on urban and suburban routes. 

## Technology Used
- Flutter and Dart: The PDS app is developed using Flutter, a popular open-source UI software development toolkit created by Google. Dart, a client-optimized language for fast apps on any platform, is used for coding.
- Firebase Cloud Firestore: Firebase Cloud Firestore, a scalable NoSQL cloud database, is used as the backend database for storing data on pothole locations.
